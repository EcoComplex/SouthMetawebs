### Model description

We consider that the compiled food webs are a representation of all posible interactions, and at any given time if a species is not present in the local site the interaction do not realize, so the number of species and interactions can fluctuate over time. To represent this network variability we used a metaweb assembly model [@Saravia2022], assuming that each compiled food web represent the metaweb for each region.

In this model species migrate from the metaweb to a local site with a probability $c$, and become extinct from the local site with probability $e$; a reminiscence of the theory of island biogeography [@MacArthur1967], but with the addition of network structure. Species migrate with their potential network links from the metaweb, then in the local site only the interactions with the present species are realized. Species have a probability of secondary extinction $se$ if none of its preys are present, which only applies to non-basal species. When a species goes extinct locally it may produce secondary extinctions modulated by $se$ .

The model is implemented in R using the package `meweasmo` [@Saravia2022b]. The 3 parameters of the model are fitted for each compiled food web using approximate Bayesian computation (ABC) [@Beaumont2002]. The ABC algorithm is implemented in the package `abc` [@abc] using the `abc` function with the `rejection` method. The distance function used is the euclidean distance between the observed and simulated number of species and connectance. The prior distributions for the parameters are uniform: $c \sim U(0.01,0.3)$, $e \sim U(0.001,0.5)$, and $se \sim U(0.1,0.9)$. The number of simulations is set to 10000, and the tolerance is set to 0.05.

### Dynamic stability

We use the maximal real part of the eigenvalues of the community matrix jacobian [@Allesina2015a] for randomly parametrized systems, keeping fixed the predator–prey (sign) structure and assuming the system is stable [@Barabas2017]. This is a measure of the dynamic stability of the system, which is defined as the ability of the system to return to equilibrium after a perturbation. The values of the entries of the jacobian are sampled from a uniform distribution, assuming that the positive effect of preys on predators have a maximal value of 1 and the negative effect of predators on preys have a minimal value of -10. The diagonal entries of the jacobian that represent the self-regulation of the species are sampled from a uniform distribution between *-SelfReg* and 0, with the value of *SelfReg* determined by simulations given that the 5% of the time the network is stable (That means that the maximal real part of the eigenvalues is negative).  To find the value of *SelfReg* we perform a grid search over the range of values from -7 to -60 in steps of 1, and for each value we sample 10000 jacobians and calculate the proportion of stable systems. Then we interpolate the results to find the value of *SelfReg* that gives a proportion of stable systems of 0.05. Finally for the value of *SelfReg* that we found we perform 10000 simulations and select only the stable ones to obtain a distribution of the maximal real part of the eigenvalues for each network. 


#### Entropy

Ecological networks are primarily represented by their adjacency matrices, i.e., a matrix in which every entry represents a pair of species, which can take a value of 1 when the two species interact, and a value of 0 when they do not. These matrices (as any matrices) can easily be factorised using Singular Value Decomposition (Forsythe and Moler, 1967; Golub and Reinsch, 1971), which offers two interesting candidate measures of complexity for ecological networks (both of which we describe at length in the methods). The first measure is the rank of the matrix, which works as an estimate of “external complexity,” in that it describes the dimension of the vector space of this matrix, and therefore the number of linearly independent rows (or columns) of it. From an ecological standpoint, this quantifies the number of unique “strategies” represented in the network: a network with two modules that are distinct complete graphs has a rank of 2. The second measure is an application of the entropy measure of Shannon (1948) to the non-zero singular values of the matrix obtained through SVD. This so-called SVD entropy measures the extent to which each rank encodes an equal amount of information, as the singular values capture the importance of each rank to reconstruct the original matrix; this approach therefore serves as a measure of “internal complexity.”



#### Estimating Network Complexity with SVD Entropy

Singular Value Decomposition (SVD) factorizes a matrix $ \mathbf{A} $ into the form
$$
\mathbf{A} = \mathbf{U} \, \boldsymbol{\Sigma} \, \mathbf{V}^T,
$$
where $ \mathbf{U} $ and $ \mathbf{V} $ are orthogonal matrices, the columns in these matrices are, respectively, the left- and right-singular vectors of $ \mathbf{A}$. $\boldsymbol{\Sigma} $ is a diagonal matrix containing the non-negative singular values $\sigma_i$ of $\mathbf{A}$ in descending order. The singular values indicate the relative weight of each structural dimension represented in the matrix. 

When SVD is applied to a mean-centered data matrix or a covariance matrix, the squared singular values are equal to the eigenvalues of the covariance matrix and thus quantify the proportion of total variance explained by each principal component [@Jolliffe2002; @Golub2013]. In the context of ecological networks, the singular values measure how strongly each independent dimension contributes to the overall network structure .

To estimate the complexity of an ecological network, the normalized singular values are used to compute the **SVD entropy**, following @Shannon1948. The normalized singular values are:
$$
s_i = \frac{\sigma_i}{\sum_{j=1}^{k} \sigma_j},
$$
where $ k $ is the number of non-zero singular values.

The SVD entropy $ J $ is then defined as:
$$
J = - \frac{1}{\ln(k)} \sum_{i=1}^{k} s_i \ln(s_i),
$$
where the normalization by $ \ln(k) $ ensures that the maximum entropy is 1, equivalent to Pielou's evenness index [@Pielou1975].

High SVD entropy indicates that structural information is evenly distributed among dimensions, implying high network complexity and low compressibility [@Strydom2021].



#### Estimating Complexity With Relative Rank Deficiency

In addition to the SVD entropy, we also estimate the complexity of ecological networks using the rank deficiency of the adjacency matrix. The rank deficiency is a measure of how many rows/columns in a matrix are not linearly independent, and therefore how many rows/columns can be expressed as linear combinations of others. In ecological terms, this means that some species or interactions can be described as a combination of others, which reduces the complexity of the network.

The rank of $A$ (noted as $k$) is the dimension of the vector space spanned by the matrix and corresponds to the number of linearly independent rows or columns; therefore, the maximum rank of a matrix $M$ will always be equal to the dimension of A, in the case of trophic networks A is square,  which ecologically speaking is the richness. A matrix is “full-ranked” when $k = M$, i.e., all of its rows/columns are unique. Matrices that are not full-ranked are called rank deficient, and we can measure rank deficiency using the relative rank deficiency to control for the difference in species richness of the different networks, expressing it as:

$D  = 1 - (r/M )$

where D = 0 means that the matrix is full-ranked, and D = 1 means that the matrix is rank deficient. This serves as a coarse estimate of complexity, as the more unique columns/rows are in the matrix, the larger this value will be.
